# Embedded C Learning

A collection of self-designed Embedded C challenges completed while learning C programming before moving to real microcontroller development.
Programs are built directly from Visual Studio Code using GCC and GDB. The focus is to develop embedded coding skills by writing clean, warning-free,
portable C99 code. All challenges have been generated by Claude AI, but coding is done by myself, primarily using Google and Youtube documentaries to
assist in my learning.

Topics covered:
- Arrays
- Pointers
- Structs
- Enums
- Dynamic memory 
- Bitwise operations
- Const usage
- Fixed-width integers and arithmetic
- Robust input parsing
